WEB

The Web (WEB) is a computer programming system created by Donald Knuth. He made it to actually apply the literate programming he proposed. Usually, when writing a program, write the program first and then use comments to explain the program's behavior later. In literary programming, how a program should work is written out in words, just like writing a literary work, and codes related to the operation are written by separating them with special symbols. And by using the web, the part related to the description is extracted as a printable document, and the source code part is extracted as compilable code. WEB consists of two programs, 'tangle' and 'weave'.
